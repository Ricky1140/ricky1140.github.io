1.2 类型

1.2.1 长度

类型 长度

1.2.2 类型转换

1.2.2.1 隐式类型转换（自动类型转换）

  • 类型兼容（数值等）
  • 长度小 -> 长度大
int num1 = 10;
float num2;
double num3;

num2 = num1;
num3 = num1;

1.2.2.2 显式类型转换（强制类型转换）

  • 类型兼容
  • 长度大 -> 长度小
float num1 = 3.5f;
double num2 = 3.1415926;

int num3 = (int)num1;
int num4 = (int)num2;

1.2.2.3 表达式转换

int a = 10;
int b = 10;
float c = 10f;

int d = a + b;	// right
int e = a + c;	// fault
float f = a + c;// right

1.2.2.4 Convert 类型转换

  • 类型不兼容（如 string 和 int）
string num1 = "3.15";
string num2 = "198";

double num3 = Convert.ToDouble(num1);
double num4 = Convert.ToInt32(num2);

Console.WriteLine(num1 + num2);	// 3.15198
Console.Writeline(num3 + num4);	// 201.15

输入需要数字时

Convert.ToInt32(Console.ReadLine());

1.3.1 逐字字符串

  • 逐字字符串 @ 会保留所有的空格和字符，而无需转义反斜杠

1.3.2 Unicode 转义字符

  • \u
  • 在文本字符串中添加编码字符，然后使用四个字符的代码表示 Unicode (UTF-16) 中的某些字符
// Kon'nichiwa World
Console.WriteLine("\u3053\u3093\u306B\u3061\u306F World!");

// To generate Japanese invoices:
// Nihon no seikyū-sho o seisei suru ni wa:
Console.Write("\n\n\u65e5\u672c\u306e\u8acb\u6c42\u66f8\u3092\u751f\u6210\u3059\u308b\u306b\u306f\uff1a\n\t");
// User command to run an application
Console.WriteLine(@"c:\invoices\app.exe -j");

1.6.4 内存结构

1.6.4.1 值类型

int, float, double, bool, char

  • 内存：值类型的值存储在内存的栈中
int a = 10;
int b = a;	// 变量间的传值

b = 20;

Console.WriteLine("{0}, {1}", a, b);

  • 这个时候单独改变 b 的值，a 的值不会改变（把 a 的值拷贝给 b）
  • 这种传递叫值传递

简单的说就是地址不是一个，所以不会变

1.6.4.2 引用类型

数组，字符串

  • 内存：引用类型的值存储在内存的堆中
int[] intA = new int[] {00,111,222,333,444};
int[] intB = intA;
  • 此时单独修改 intb 中元素的值，inta 的值会发生改变
  • 这种传递叫引用传递

初始化 intA时：

  • 在栈中创建 intA
  • 在堆中创建 new int[]
  • 把上述 new int[] 的内存赋值给 intA

intB = intA 就相当于 intB 的值是 intA 的值的拷贝，但 intA 的值是内存，所以更改 intB 也会更改 intA

因为地址里的东西改了，他俩指向同一个地址

1.6.4.3 数组的内存结构

  • 数组在内存中是一块连续的存储空间存储的
  • 最低的地址对应第一个元素，最高的地址对应最后一个元素

1.7.1 函数重载

static void Main(string[] args)
{
    // 定义一个 Add 方法，实现整数、小数相加操作
    Console.WriteLine(Add(1, 2));
    Console.WriteLine(Add(3.1234, 8.5678));
    Console.WriteLine(Add(3, 3.14));
}

static int Add(int a, int b)
{
    return a + b;
}

static double Add(double a, double b)
{
    return a + b;
}

static double Add(int a, double b)
{
    return a + b;
}
  • 参数类型一样：参数数量不一样
  • 参数数量一样：参数类型不一样
  • 返回值和重载没关系

1.7.2 高级参数

1.7.2.1 普通调用

using System;

namespace FunctionParam
{
    class Program
    {
        static void Main(string[] args)
        {
            // 演示：Add(int num)：将一个值自身加10
            int num = 5;
            Add(num);

            Console.WriteLine(num);	// 5
        }

        static void Add(int num)
        {
            Console.WriteLine("num_start: " + num);	//5
            num += 10;
            Console.WriteLine("num_end: " + num);//15
        }
    }
}

1.7.2.2 ref 参数

  • 作用
    • 将一个变量传入一个函数中进行 “处理”，“处理” 完成后，再将 “处理” 后的值带出函数。
  • 要求
    • 函数外必须为变量赋值，而函数内可以不赋值
  • 语法
    • 形参和实参前面都要加 ref 关键字
static void Main(string[] args)
{
    // 演示：Add(int num)：将一个值自身加10
    int num = 5;
    Add(ref num);

    Console.WriteLine(num);	//15
}

static void Add(ref int num)
{
     Console.WriteLine("num_start: " + num);//5
     num += 10;
     Console.WriteLine("num_end: " + num);	//15
}
  • 内存原理

1.7.2.3 out 参数

  • 作用
    • 一个函数中如果返回多个不同类型的值，就需要用到 out 参数
  • 要求
    • 函数外可以不为变量赋值，而函数内必须为其赋值
  • 语法
    • 形参和实参前面都要加 out 关键字

3.7 [array].GetLength()

⭐与 Length 的区别

  • 假设这里有个数组：
int[,] array = new int[,] {
    {1,2,3},
    {4,5,6}
};

int a = array.Length;
int b = array.GetLength(0);
int c = array.GetLength(1);

Console.WriteLine(a);
Console.WriteLine(b);
Console.WriteLine(c);
  • 运行结果如下：

结论：

  • Length 不需要括号
  • Length = 行 * 列
  • GetLength(0) = 行数
  • GetLength(1) = 列数
