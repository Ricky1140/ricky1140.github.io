洛谷P1572 Java解法

2023-01-30 数据结构与算法 洛谷 1.1k 字
最大公因数 最小公倍数

P1572 计算分数 - 洛谷 | 计算机科学教育新生态 (luogu.com.cn)

以上是题目链接。

这个题吧，让我很生气。

其实思路很简单，就是中间我想整点花活没整出来导致做了三个点（恼）

当时还去看了一下题解想参考参考然后都是cpp的（悲）

最后输入那里整不出其他活就只能用古老的循环替换哩

那么接下来是题解

这个题目的要求是计算分数的加减运算，并且要化最简。

那么我们可以很容易想到：

通分时求最小公倍数，化简时求最大公因数

于是就有了以下代码：

// 最大公因数
public static int gcd(int m, int n) {
	if (n == 0) return m;
	else return gcd(n, m % n);
}

然后还有一个公式：

最小公倍数 = 两整数乘积 / 最大公因数

public static int lcm(int m, int n) {
	return m * n / gcd(m, n);
}

接收数字的时候稍微麻烦一点，c语言和cpp可以用 scanf("%d/%d", &a, &b) 来解决，甚至少了判断符号这一步，但是我们在用 java（悲）

所以这里还是用 sc.nextLine() 接收一整个字符串。

在这里，我用了个 List<Character> 存储符号，本来也想用类似的方法存储数字但有些数字在两位以上，就会被分开导致计算错误。

去搜了一下 Scanner 有没有其他用法的时候看见了这样的用法：

String input = "1 fish 2 fish red fish blue fish";
Scanner s = new Scanner(input).useDelimiter("\\s*fish\\s*");
System.out.println(s.nextInt());
System.out.println(s.nextInt());
System.out.println(s.next());
System.out.println(s.next());
s.close();

输出结果为：

1
2
red
blue

这里是用 Scanner 设置特定的分隔符，这样就能分开 / 前后的数字了，a是存储分子的 List，b存储分母

Scanner s = new Scanner(sb.toString()).useDelimiter("\\s*/\\s*");
		
while (s.hasNextInt()) {
	a.add(s.nextInt());
	b.add(s.nextInt());
}

这里就分好分子和分母了，接下来求分母的最小公倍数，通分后计算分子，然后得出的结果再求分子和分母的最大公因数。分子分母同除最大公因数。这里因为结果输出是横向的，所以要判断一下分母是否为负，如果为负则分子分母同时变号。

输出的时候直接输出分子，如果分子不能整除分母的话再输出 / 和分母。

以上就是整道题的思路。

以下是完整源代码：

import java.util.Scanner;
import java.util.List;
import java.util.ArrayList;

public class P1572 {
	public static void main(String[] args) {
		Scanner sc = new Scanner(System.in);
		
		StringBuffer sb = new StringBuffer();
		
		List<Integer> a = new ArrayList<>();
		List<Integer> b = new ArrayList<>();
		List<Character> c = new ArrayList<>();
		
		int lcm = 1;
		int gcd = 1;
		
		// 用字符串接收，将分子和分母分别存
		String in = sc.nextLine();
		
		if (in.charAt(0) != '-') {
			c.add('+');
		}
		
		for (int i = 0; i < in.length(); i++) {
			if (in.charAt(i) == '-' || in.charAt(i) == '+') {
				c.add(in.charAt(i));
			}
		}
		
		for (Character ch : in.toCharArray()) {
			if (ch == '+' || ch == '-') {
				sb.append('/');
			} else {
				sb.append(ch);
			}
		}
		
//		System.out.println(sb.toString());
		
		Scanner s = new Scanner(sb.toString()).useDelimiter("\\s*/\\s*");
		
		while (s.hasNextInt()) {
			a.add(s.nextInt());
			b.add(s.nextInt());
		}
//		
//		System.out.println();
//		System.out.println(a.toString());
//		System.out.println();
//		System.out.println(b.toString());
//		System.out.println();
//		System.out.println(c.toString());
		
		for (int i = 0; i < b.size(); i++) {
			lcm = lcm(lcm, b.get(i));
		}
		
		for (int i = 0; i < a.size(); i++) {
			a.set(i, a.get(i) * (lcm / b.get(i)));
		}
		
		int ansa = 0;
		int ansb = lcm;
		
		for (int i = 0; i < c.size(); i++) {
			if (c.get(i) == '+') {
				ansa += a.get(i);
			} else {
				ansa -= a.get(i);
			}
		}
		
		gcd = gcd(ansa, ansb);
		
		ansa /= gcd;
		ansb /= gcd;
//		
//		System.out.println(ansa);
//		System.out.println(ansb);
//		System.out.println(gcd);
//		System.out.println(lcm);
//		
		
		if (ansb < 0) {
			ansa = -ansa;
			ansb = -ansb;
		}
		
		System.out.print(ansa);
		
		if (ansa % ansb != 0) {
			System.out.print("/" + ansb);
		}

		sc.close();
	}
	
	// 最小公倍数 = 两整数乘积 / 最大公因数
	public static int lcm(int m, int n) {
		return m * n / gcd(m, n);
	}
	
	// 最大公因数
	public static int gcd(int m, int n) {
		if (n == 0) return m;
		else return gcd(n, m % n);
	}
}
