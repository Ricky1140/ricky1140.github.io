洛谷P1115 Java解法

2023-01-31 数据结构与算法 洛谷 304 字
dp

P1115 最大子段和 - 洛谷 | 计算机科学教育新生态 (luogu.com.cn)

完全不会动态规划所以直接抄的题解（悲）

所以这一篇来分析代码吧！

题目写得挺诱导人的，我最开始一直在想怎么才能取区间，但其实根本不需要区间只需要最大值（恼）

这里我们使用边读边算的做法。

n 是用来接第一个数的，同时也是循环条件。

a 用来每轮循环读一个数进行计算。

ans 是要输出的结果，设置初始值为 1 << 31 的原因是中间有个测试点是负数，不知道位运算是怎么算的直接移步到算法新手班 01 课的笔记吧

新手班-01笔记

now 用来记录现在的子段和，并和 ans 比较谁更大。

以下为完整代码：

import java.util.Scanner;

public class P1115 {
	public static void main(String[] args) {
		Scanner sc = new Scanner(System.in);
		
		int n;
		int a;
		int ans = 1 << 31;
		int now = 0;
		
		n = sc.nextInt();
		
		while (n-- != 0) {
			a = sc.nextInt();
			now += a;
			if (now > ans) ans = now;
			if (now < 0) now = 0;
		}
		
		System.out.println(ans);
		
		sc.close();
	}
}
