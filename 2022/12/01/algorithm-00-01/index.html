⛄二进制与位运算

  • 计算机内部存储中，int类型为32位，long类型为64位。

假如，有一个int型在存储中为：

​ 00000000 00000000 00000000 00000001

那么，1做向左移动31为的位运算结果为：

​ 10000000 00000000 00000000 00000000

此时，num与1左移 i 位（i 最高为31，最低为0）的结果的 &（与）运算为：

​ 1 左移后，右边 i - 1 位都为 0，所以结果看 1 和 num 的从右向左第 i 位的与运算结果

Q：你能输出一个int类型的32位吗？

public class Code06_PrintB {
	
	public static void print(int num) {
		for (int i = 31; i >= 0; i--) {
			System.out.print( (num & (1 << i)) == 0? "0" : "1");
		}
		System.out.println();
	}
	
	public static void main(String[] args) {
		// 32位		long 64位
		int num = 83928328;
		
		print(num); 
	}
}

左移

  • 如果是整个数进行左移那就整体往左移，右边补零。
  • num << 1 -> num *= 2;
  • int 第 1 位作为符号，第 2 位到 32 位为数字。
    • 符号位为 0 时，是非负数，值为数值位。
    • 符号位为 1 时，是负数，值为数值位取反+1。
      • 加减乘除在底层中都是二进制，为了计算正确所以负数是取反+1

比如，-1 为：

​ 11111111 11111111 11111111 11111111

其中，第一位为符号位，代表负。

值为除符号位外取反后加一，即：

​ 10000000 00000000 00000000 00000000

后面为0，加一即为1。

  • Java 中没有无符号int。范围：-2^31^~2^31^-1
  • Java 中取最大最小值如下：
int a = Integer.MAX_VALUE;
int b = Integer.MIN_VALUE;

最大值二进制为：

​ 01111111 11111111 11111111 11111111

最小值二进制为：

​ 10000000 00000000 00000000 00000000

  • 取反符号如下：
int a = 123456789;
int b = ~a;
  • 与、或、异或符号如下：
int a = 12345678;
int b = 87654321;
// 调用上面写的print函数
print(a | b);	// 或，有1得1
print(a & b);	// 与，都1得1
print(a ^ b);	// 异或，相同为0，不同为1

右移

int c = Integer.MIN_VALUE;
print(c);
print(c >> 1);	// 带符号右移
print(c >>> 1);	// 不带符号右移

带符号右移时，移动后正数用 0 填补，负数用 1 填补

比如以上代码的运行结果为：

    10000000000000000000000000000000
    11000000000000000000000000000000
    01000000000000000000000000000000

相反数

int d = 5;
int e = -d;
int f = ~d + 1;
		
System.out.println(d);
System.out.println(e);
System.out.println(f);

运行结果：

5
-5
-5

5 的二进制为：

​ 00000000 00000000 00000000 00000101

取反后：

​ 11111111 11111111 11111111 11111010

+1 后：

​ 11111111 11111111 11111111 11111011

翻译：（最高位为 1 代表负，然后取反 +1）

​ - 0000000 00000000 00000000 00000101

即为 -5

  • 即负号可以用取反加一代替
  • 原数为正负都对，越底层越高效
  • 当整型最小值取相反数时，会得到整型最小值（即自己）
  • 0 取相反数时二进制取反溢出，溢出后不要，所以相反数还是0

完整代码：

public class Code06_PrintB {
	
	public static void print(int num) {
		for (int i = 31; i >= 0; i--) {
			System.out.print( (num & (1 << i)) == 0? "0" : "1");
		}
		System.out.println();
	}
	
	public static void main(String[] args) {
		// 32位		long 64位
		int num = 83928328;
		
		print(num); 
		print(~num);
		
		System.out.println();
		
		int a = 12345678;
		int b = 87654321;
		
		print(a | b);	// 或，有1得1
		print(a & b);	// 与，都1得1
		print(a ^ b);	// 异或，相同为0，不同为1
		
		System.out.println();
		
		// 右移
		int c = Integer.MIN_VALUE;
		print(c);
		print(c >> 2);	// 带符号右移
		print(c >>> 1);	// 不带符号右移
		
		System.out.println();
		
		// 相反数
		int d = 5;
		int e = -d;
		int f = ~d + 1;
		
		System.out.println(d);
		System.out.println(e);
		System.out.println(f);
		
		System.out.println();
		
		int g = Integer.MIN_VALUE;
		int h = -g;
		int i = ~g + 1;
		
		System.out.println(g);
		System.out.println(h);
		System.out.println(i);
	}
}

🌊选择排序

感觉没啥好讲的直接放代码好了

public class code07_SelectionSort {
	public static void selectSort(int[] arr) {
		if (arr == null || arr.length < 2)
			return;
		int N = arr.length;
		
		for (int i = 0; i < N; i++) {
			int minValueIndex = i;
			for (int j = i + 1; j < N; j++) {
				minValueIndex = arr[j] < arr[minValueIndex] ? j : minValueIndex;
			}
			swap(arr, i, minValueIndex);
		}
	}
	
	public static void swap(int[] arr, int i, int j) {
		int tmp = arr[j];
		arr[j] = arr[i];
		arr[i] = tmp;
	}

	public static void printArray(int[] arr) {
		for (int i = 0; i < arr.length; i++) {
			System.out.print(arr[i] + " ");
		}
		System.out.println();
	}

	public static void main(String[] args) {
		int[] arr = { 7, 1, 3, 5, 1, 6, 8, 1, 3, 5, 7, 5, 6 };
		printArray(arr);
		selectSort(arr);
		printArray(arr);
	}
}

🌀冒泡排序

  • 相邻两个数谁大谁往后走

这里就只写一个函数了，swap 调用的是上面那个

public static void bubbleSort(int[] arr) {
	if (arr == null || arr.length < 2)
		return;
	int N = arr.length;
	
	for (int i = N - 1; i >= 0; i--) {
		for (int j = 0; j < i; j++) {
			if (arr[j] > arr[j + 1]) {
				swap(arr, j, j+1);
			}
		}
	}
}

第二层循环也可以设置成第二个数：

public static void bubbleSort(int[] arr) {
	if (arr == null || arr.length < 2)
		return;
	int N = arr.length;
	
	for (int i = N - 1; i >= 0; i--) {
		for (int j = 1; j <= i; j++) {
			if (arr[j - 1] > arr[j]) {
				swap(arr, j-1, j);
			}
		}
	}
}

🔥插入排序

  • 第 i 位的数向左看，比左边的小就交换，直到前 i 位有序

同样只写个函数：

public static void insertSort(int[] arr) {
	if (arr == null || arr.length < 2)
		return;
	int N = arr.length;
	
	for (int i = 1; i < N; i++) {
		int newNumIndex = i;
		while (newNumIndex - 1 >= 0 && arr[newNumIndex-1] > arr[newNumIndex]) {
			swap(arr, newNumIndex-1, newNumIndex);
			newNumIndex--;
		}
	}
}

优化版：

public static void insertSort2(int[] arr) {
	if (arr == null || arr.length < 2)
		return;
	int N = arr.length;

	for (int i = 1; i < N; i++) {
		for (int j = i - 1; i >= 0 && arr[j] > arr[j + 1]; j--) {
			swap(arr, j, j + 1);
		}
	}
}
